Qn56. Define a harddisk interface

A hard disk interface defines the physical means by which the hard disk connects to the rest of the computer.

Qn57. State any three internal harddisk interfaces

  • External hard disk interfaces include:
    • USB
    • FireWire
  • Internal hard disk interfaces Include:
    • SATA, (Serial Advanced Technology Attachment)
    • EIDE, (Enhanced Integrated Drive Electronics)
    • SCSI, (Small Computer System Interface) and
    • SAS (Serial-attached SCSI)

Qn58. Define a solid-state storage media

Solid state is a non-volatile storage that employs integrated circuits rather than mechanical, magnetic or optical technology
