Qn11. List any five functions of an operating system

• Starting a computer, (Booting the computer)
• Providing a user interface
• Managing programs
• Configuring devices
• Monitoring performance
• Providing file management
• Administering security
• Managing resources
• Coordinating tasks, and Spooling
• Managing memory
• Establishing network connections

Qn12. List any five factors to consider before choosing an operating system

  • The type of computer in terms of size and make
  • The hardware configuration of the computer
  • Adequate information and help guides for user reference
  • The application software to be installed
  • User friendliness
  • The cost of the operating system
  • Reliability and security provided by the operating system
  • The number of processors and hardware devices it can support
  • The number of users it can support
  • The availability of basic utilities and accessory programs within the operating system

Qn14. Define device drivers

A device driver is a program that enables a computer to recognise and effectively communicate with a given hardware

Qn17. Define a language translator

Qn18. Explain the following terms as used in programming

a. Compiler

A compiler is a computer program that translates the entire source code at once into an object code

b. Assembler

a program for converting instructions written in low-level symbolic code into machine code.

c. Interpreter

A computer program that translates source code into object code line by line and executes it

d. Linker

A linker or link editor is a computer system program that takes one or more object files (generated by a compiler or an assembler) and combines them into a single executable file, library file, or another "object" file.
