SubTopic 1: System Software

Qn1. Define the term Computer Software

Computer software refers to the electronic instructions and procedures that control the operation of a computer

Qn2. State the two major types of software

System software and application software

Qn3. Differentiate between Application software and System software

System software e.g., the Operating system manage and coordinate all the other computer programs, devices, resources and activities WHILE Application software like, Word-processors, Spreadsheets, Media players and Games solve the specific or exact needs of the user

Qn4. Explain the three categories of system software

  • An operating system is a generalized program that manages and coordinates all the activities taking place within a computer system
  • Utility software refers to system software designed to analyze, optimize, enhance and maintain a computer in good working conditions.
  • A programming language is a notation for writing computer software

Qn8. Define firmware

This refers data, instructions, or programs permanently and collectively written to a ROM chip with in a device by its manufacturers

Qn9. Briefly explain the following types of operating system stating an example in each case

a. Stand-alone OS

A stand-alone operating system is a complete operating system that works on a PC e.g., Mac OS X, UNIX, Linux, MS-DOS and Windows (XP, Windows Vista, Windows 7, Windows 8, Windows 10 etc.).

b. Network OS

A network operating system (NOS) is an operating system that supports a network and typically resides on the server e.g., Windows Server 2003, 2008, Solaris, and NetWare, UNIX server, and Linux server

c. Embedded OS

An embedded operating system is an operating system that resides on ROM chips and typically used on handheld computers and small devices e.g., Windows Embedded CE, Windows Mobile, Palm OS, Embedded Linux, and Symbian OS

Qn10. What is meant by

a. Proprietary

Proprietary software is privately owned and limited to a specific computer model

b. Device-dependent

A device-dependent program is one that runs only on a specific type or brand of computer
