SubTopic 1: Introduction to Computers

Qn1. Define a computer

A computer is an electronic device which accepts data, processes it under special instructions, stores and outputs the information OR
A computer is an electronic artificially intelligent machine that is capable of performing different tasks according to instructions that are stored internally either temporarily or permanently.

Qn2. What is a computer system?

A computer system is a collection of interrelated computer components working together to input, output, process, and store data and information.

Qn3. State and explain the five components of a computer system

  1. Computer Hardware: This is a term used to describe all the various physical/tangible devices of a computer.
  2. Computer Software: Software is a term for electronic instructions that tell the computer how to perform a task.
  3. Computer Users (Humanware/Liveware): refer to the people who operate and initialize instructions to the computer system.
  4. Data refers to raw facts and figures used to create information
  5. Computer communication is the transmission of data and information over a channel between two computing devices.

Qn4. Differentiate between ordinary user and professional user

An Ordinary user - is someone without much technical knowledge of computers but uses computers to produce information for professional or personal tasks, enhance learning, or have fun WHILE A Professional user -is a person in a profession involving computers who has had formal education in the technical aspects of computers

Qn5. Differentiate between data and information

Data refers to raw facts and figures used to create information WHILE Information refers to the processed data that makes meaning and is useful.

Qn6. State any four characteristics of meaningful information

  • Timely −Information should be available when required.
  • Accuracy −No possibilities of even slightest errors
  • Completeness −Information should be complete.
  • Comprehensive –should offer all the desired explanations

Qn7. Define an information processing cycle

The information processing cycle is a series of steps the computer follows to process data

Qn8. With the aid of a well labelled diagram describe the information processing cycle

  • Input: The computer accepts data from some source
  • Processing: The computer's processing components perform actions on or with the data
  • Output: The computer provides the results of its processing
  • Storage: The computer stores the results of its processing.
