Qn16. Define a virus signature/virus definition

Specific pattern of virus code

Qn17. Suggest any two actions taken by an antivirus program after identifying an infected file in a computer

  • Attempts to remove any detected virus
  • Quarantines infected files that it cannot remove

Qn18. Suggest any five control measures against viruses

  • Install the latest versions of anti-virus software on the computers
  • Always scan removable storage media for viruses before using them
  • Scan mail attachments for viruses before opening or downloading an attachment
  • Always keep a Recovery Disk
  • Set the macro security in programs so you can enable or disable macros
  • Check all downloaded programs for viruses, worms, or Trojan horses
  • Install a personal firewall program

Qn19. Suggest any three ways computer viruses are spread on a standalone computer

  • Distributed through flash disks
  • By using floppy diskettes
  • Through opening infected programs or documents on CR/DVD discs

Qn20. Suggest any three ways computer viruses are spread on a networked computer

  • Through downloading email attachment
  • Playing games on internet
  • Downloading infected files from internet
