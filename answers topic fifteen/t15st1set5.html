Qn21. Differentiate between hacking and cracking

Hacking is the unauthorised access to or control over computer network security systems for some illicit purpose WHILE Cracking is when someone performs a security hack for criminal or malicious reasons

Qn22. Briefly describe the following in relation to hacking

a. A script kiddie

A script kiddie has the same intent as a cracker but does not have the technical skills and knowledge

b. Cyber extortionist

A cyberextortionist is someone who uses e-mail as a vehicle for extortion.

c. Cyberterrorist

A cyberterrorist is someone who uses the Internet or network to destroy or damage computers for political reasons.

Qn23. Briefly explain the following

a. Denial of service attack

A denial of service attack, or DoS attack, is an assault whose purpose is to disrupt computer access to an Internet service

b. Botnet

A botnet is a group of compromised computers connected to a network such as the Internet that are used as part of a network that attacks other networks, usually for nefarious purposes.

c. Zombie

This is a compromised computer whose owner is unaware the computer is being controlled remotely by an outsider.

d. Backdoor attack

A back door is a program or set of instructions in a program that allow users to bypass security controls when accessing a program, computer, or network

e. Spoofing

Spoofing is a technique intruders use to make their network or Internet transmission appear legitimate to a victim computer or network
