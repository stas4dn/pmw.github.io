SubTopic 2: File Management

Qn8. List any five file management operations

  • Creating files and folders
  • Moving
  • Copying
  • Selecting single or multiple files (ctrl+click / Shift+click)
  • Renaming
  • Searching and Sorting by name, date or file extension (type)
  • Printing
  • Deleting
  • Restoring or emptying recycle bin etc.

Qn9. Define a folder

A Folder/Directory is a named storage location where related files can be stored. OR
A Folder is a virtual container for files and other folders.

Qn10. State any four features of a folder

  • A folder has a name
  • A folder has a path originating from a special directory called root directory
  • A folder has special access permissions for authenticated users
  • A folder or directory may be created inside another folder or directory

Qn11. Define a subfolder

Refers to a folder or directory created inside another folder or directory

Qn12. As a computer student of IGASCO advise Mr Balikweya on how to;

a. Create a folder on the desktop

  • Right click in the location where you would like to place the folder
  • Select NEW
  • Select Folder
  • Or
    CTRL+Shift+N

b. Rename a folder

  • Right Click on folder and choose Rename option from the popup menu.
    Or Select the file or folder icon and Press F2 function key,
    Or Click the folder name twice (note: not double clicking)
  • Type the new name, and then press Enter key

c. Move a folder

  • Right click on the folder and click Cut.
  • Right click the drive or folder where you want the item moved and click paste.
  • Or
    Drag and drop to the desired location

d. Delete a folder

  • Right click the folder and Select delete from the popup menu Or
  • Select the file and Press delete key on the keyboard or Press Shift+Delete on the keyboard to permanently delete the file
    NB: Deleting a folder or file: Warning: Items deleted from a removable storage are not held in the recycle bin and are completely lost.

Qn13. Define a file

A file is a collection of related data or information stored under a given name with a given file extension

Qn14. Define a file extension

A suffix(string of characters attached at the end of a filename), usually preceded by a full stop and indicating the format of the file

Qn15. List any three examples of computer files

  • Every document
  • picture
  • sound
  • video etc.

Qn16. Explain the steps required to create a new text file on the desktop

  • On the free space on the desktop, right click.
  • Point new, click Text Document. From the list available on application.
  • Type a new name for the new file to replace the temporary name and press Enter key.
